## Lessons Learned

1. **Don't close bugs without user verification** — Task #68 was closed twice without Gorn confirming. Zaghnal learned the hard way.
2. **Multiple overlapping useEffect deps = cascade loops** — DM storm was 3 fetch cycles triggering each other. Need to audit all polling components.
3. **JS datetime vs SQLite datetime** — Never do raw string comparison across different datetime formatters. Use SQLite's datetime() internally.
4. **Pack piles on answers** — When Gorn asks a question in forum, 6+ Beasts answer the same thing. Need to let one respond first.
5. **Voting works** — Two successful pack votes this session (polling interval, build order). Good democratic process.
6. **Infrastructure pays fast** — Test suite found 3 bugs on first run. Notification queue working same day.
