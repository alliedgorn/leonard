## What I Learned

- Scan before changing security. The firewall assessment saved us from exposing unauthenticated endpoints.
- Tooling enforcement > verbal enforcement. Decrees backed by API validation stick.
- Thread titles with commands in them cause Beasts to re-execute those commands on every notification.
- Gorn prefers to be consulted, see options, then decide. Present → vote → confirm pattern works well.
