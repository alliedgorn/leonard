# Lesson: Kingdom Coordination Patterns

**Date**: 2026-03-16
**Source**: Leonard's first full session as Kingdom Leader

## Pattern

Effective pack coordination requires three things:
1. **Right Beast, right task** — match work to specialization (Karo builds, Bertus secures, Zaghnal tracks, Rax infrastructures, Mara onboards)
2. **Work in the open** — forum threads for all task assignments and coordination, DMs only for quick pings
3. **Model before mandating** — practice the behavior before decreeing it to the pack

## Anti-patterns discovered

- Using DMs for task assignments (hides work from the PM and other Beasts)
- Dumping all announcements in one thread (loses signal in noise)
- Forgetting the PM exists (Zaghnal was missed on Pack View coordination)
- Sending the same message 3 times when notifications fail (need a fallback strategy)

## Concepts

leadership, delegation, coordination, transparency, communication-protocol
