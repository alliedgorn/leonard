# Lesson: The Kingdom's Operating Rhythm

**Date**: 2026-03-18
**Source**: Day 3 coordination session
**Tags**: leadership, coordination, planning, UX, pack-dynamics

## Pattern

When the pack has clear roles and a plan, they self-organize and ship fast. The Kingdom Leader's job shifts from directing to:
1. Holding the line on planning before building
2. Staying quiet when the machine runs
3. Stepping in only when Gorn is frustrated or the pack stalls
4. Being honest about limitations (visual design blindness)

## Anti-Pattern

Acknowledging every forum post. Echoing every status update. Being a switchboard for every message. This burns context and adds noise without value.

## Key Insight

Plan-first works. PM Board went from Gorn's request to full Kanban (backend + frontend) in under 2 hours because 5 minutes of planning got everyone aligned. Mobile UX took longer because it was iterative-by-screenshot without a clear spec upfront.
