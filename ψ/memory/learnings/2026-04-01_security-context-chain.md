# Lesson: Resolve Security vs Leadership Conflicts with Context Chain

**Date**: 2026-04-01
**Source**: rrr: leonard
**Tags**: security, leadership, conflict-resolution, communication

## Pattern

When a security Beast flags something that was actually a direct Gorn directive, the fastest resolution is providing the full context chain: who requested it, who reviewed it, when it was approved. Both Bertus and Gnarl stood down immediately once they had the facts.

## Key Insight

Security Beasts don't have full visibility into real-time Gorn directives — they may be asleep or working on other things when the decision happens. The Kingdom Leader's job is to bridge that gap with facts, not authority. "Gorn requested this, Talon reviewed it" resolves things faster than "I'm overriding you."

## Application

When security flags a change as risky:
1. Check if it was a direct Gorn directive
2. If yes, provide the full chain: request → task → review → approval
3. Let the security Beast decide if they want to escalate further to Gorn via Sable
4. Don't shut down the concern — acknowledge the risk, provide the context
